# Null Watch 
# 🕒 Vintage Watch Inventory & Finance Management System

A comprehensive web-based application designed to manage vintage watch inventories, track business financial reports, and streamline data handling with seamless Excel/PDF integration.

## 🚀 Features

### 📦 Inventory Management
*   **Track Unique Artifacts:** Manage stock for vintage watches, including specific details like serial numbers, brand, movement type, condition, and year of production.
*   **Real-time Stock Alerts:** Automated notifications for low-stock items or high-demand pieces.

### 💰 Financial Reporting
*   **Automated Ledgers:** Track capital, sales revenue, profit margins, and operational expenses.
*   **Financial Insights:** Generate clean, comprehensive financial statements tailored for small to medium luxury watch traders.

### 📊 Data Integration & Export
*   **Bulk Import:** Save time by importing large collections of watch data directly via **Excel (.xlsx)** templates.
*   **Flexible Exports:** Generate and download your inventory and financial logs instantly into beautifully formatted **Excel sheets** or presentation-ready **PDF reports**.

---

## 🛠️ Tech Stack

*   **Backend:** Laravel (PHP)
*   **Database:** MySQL / PostgreSQL
*   **Frontend:** Tailwind CSS / Bootstrap & JavaScript
*   **Libraries Used:**
    *   `maatwebsite/excel` (for Excel Import & Export)
    *   `barryvdh/laravel-dompdf` or `snappy` (for PDF Generation)
